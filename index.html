How To Read The Controls
`View`: switch between stored month-end snapshots and the latest daily current view.
`Month`: choose the historical snapshot date when `Historical` is selected.
`Universe` and `Bucket`: define the comparable cohort being ranked.
`Timeframe`: changes the headline bubble map and summary emphasis.
`Threshold`: highlights the nearest move-up opportunities.
`History Window`: controls how many months appear in the drilldown chart.
`Return Source`: swaps between adjusted, distribution-adjusted, and raw return series already computed in Python.
